Fonctionnalités Principales

  • A l’enregistrement d’un nouveau contact, le système génère automatiquement un numéro selon des séquences prédéfinies et paramétrables.
    NOTE : Ce numéro peut être considéré comme le compte auxiliaire, utilisé en comptabilité.

Implémentation

  • Création d’une séquence propre aux partenaires de types clients
  • Création d’une séquence propre aux partenaires de types fournisseurs
    • Si le contact est à la fois client et fournisseurs, le système utilise la première séquence.
    • Si le contact n’est ni client ni fournisseur, le système n’utilise aucune des deux séquences.

  • L’activation de l’une des séquences génère un Numéro
    • Ce numéro est placé dans un champ nommé « Compte tiers »
    • Le champ « Compte tiers » se trouve sur la fiche contact, sous l’onglet facturation.

  • L’utilisateur peut voir ce champ s’il possède le droit suivant :
    • Montrer les fonctions de comptabilité complète
    • Voir l’onglet SREENSHOTS : Rendre le champ « Compte tiers » Visible

  • L’utilisateur peut modifier à tout moment le contenu du champ « Compte tiers »
    • Si le champ « Compte tiers » est déjà renseigné, le système ne le modifie plus automatiquement.

Démonstration De la Fonctionnalité


Rendre le champ « Compte tiers » Visible
